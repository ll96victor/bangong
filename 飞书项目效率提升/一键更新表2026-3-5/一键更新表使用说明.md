# 一键更新表工具 v1.7 使用说明

## 1. 这是什么工具

本工具用于批量提取飞书项目详情页的信息，自动整理成 CSV / Excel 文件，替代原来"手动逐条打开 → 油猴脚本提取 → 手动粘贴"的重复操作。

支持一次处理 1 ~ 60 条飞书项目链接。

---

## 2. 运行前准备

1. **安装 Python 3.9 及以上版本**（已安装可跳过）。
2. **安装依赖**：在脚本目录下运行 `pip install -r requirements.txt`
3. **准备链接**：通过油猴脚本"列表链接提取"功能，获取飞书项目链接列表。

---

## 3. 运行步骤（两步走）

### 第一步：启动 Chrome 调试模式

1. 双击 `启动Chrome调试模式.bat`
2. 会弹出一个新的 Chrome 窗口
3. **第一次使用**：在这个 Chrome 窗口中登录飞书
4. 登录完成后，Chrome 保持打开状态

### 第二步：运行脚本

1. 双击 `运行脚本.bat`
2. 选择链接输入方式
3. 等待脚本自动处理
4. 完成后查看 `output` 文件夹里的结果文件

---

## 4. 三种链接输入方式

### 方式 1：txt 文件路径

1. 将所有链接保存到一个 `.txt` 文件中，每行一个链接。
2. 输入该 `.txt` 文件的完整路径，按回车。

### 方式 2：直接粘贴链接

1. 选择选项 2
2. 粘贴链接（**提示：粘贴后链接可能显示在一行，但程序会正确识别**）
3. 输入空行结束

### 方式 3：直接按 Enter 退出

---

## 5. 新增功能

### 自动下载 chromedriver

程序会自动检测 Chrome 版本，并下载对应的 chromedriver：

1. 首次运行时会自动检查 chromedriver 是否存在
2. 如果不存在或版本不匹配，自动从官网下载
3. 下载位置：`%LOCALAPPDATA%\chromedriver_feishu\`
4. **无需用户手动操作**

### 提取失败自动重试

如果提取失败，脚本会：
1. 等待 20 秒
2. 刷新页面
3. 再等待 10 秒
4. 重新尝试提取

---

## 6. 导出文件在哪里

所有输出文件保存在脚本目录下的 `output` 文件夹。

导出字段顺序：`项目名称 | 状态 | 项目名称2 | 空 | 日期 | 来源 | aihelp链接 | 处理信息`

---

## 7. 另一台电脑 git pull 后如何使用

### 需要的额外配置

1. **安装 Python 3.9+**
2. **安装依赖**：
   ```bash
   cd 一键更新表2026-3-5
   pip install -r requirements.txt
   ```

3. **首次运行**：
   - 程序会自动检测 Chrome 版本
   - 自动下载对应的 chromedriver（如果不存在或版本不匹配）
   - **无需用户手动操作**

## 9. 常见问题

| 问题 | 解决办法 |
|------|----------|
| "无法连接到调试端口 9222" | 先运行 `启动Chrome调试模式.bat` |
| 提取的项目名称为空 | 脚本会自动重试（等待20s + 刷新 + 等待10s） |
| chromedriver 版本不匹配 | 程序会自动下载，无需手动操作 |
| 粘贴链接显示在一行 | 正常现象，程序会正确识别多个链接 |
