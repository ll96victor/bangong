## **一、需要注册但完全免费的官方翻译 API（最稳定）**

这类方案需要去平台注册并申请一个 API Key，但对于每天几十次的调用量来说，免费额度绝对绰绰有余，而且稳定性最好。

**火山翻译（字节跳动旗下）** 是目前免费额度最慷慨的选择之一，每月提供 **200万字符免费额度**，超出后按 49元/百万字符计费。接入方式是标准 REST API，支持 116 种语言，国内访问完全无障碍。对于每天几十次翻译的用量，一个月消耗的字符数连免费额度的零头都不到。官网：[火山引擎机器翻译](https://www.volcengine.com/product/machine-translation)

**腾讯翻译君** 的免费额度更大，每月高达 **500万字符免费**，同样是标准 REST API，国内访问流畅。适合对翻译质量有一定要求、希望长期稳定使用的场景。 [腾讯云机器翻译](https://cloud.tencent.com/product/tmt)

**百度翻译** 是老牌选择，个人认证后每月有 **100万字符免费额度**，文档和社区资源最丰富，网上能找到大量现成的 Python/JS 调用示例，上手最容易。 [百度翻译开放平台](https://fanyi-api.baidu.com/) [CSDN](https://blog.csdn.net/weixin_41287260/article/details/155197304)

## **二、完全无需注册的方案（零门槛但有风险）**

**DeepLX** 是一个开源项目，通过逆向工程 DeepL 的网页端接口，将其封装成本地 REST API，**无需任何 Token 或注册**，翻译质量接近 DeepL 官方水准。你只需下载对应平台的可执行文件（Windows 有 .exe），本地运行后就会在 `127.0.0.1:1188` 暴露一个 API 接口，脚本直接 POST 请求即可调用。

这个方案的优点是零门槛、翻译质量好；缺点是属于非官方逆向，如果请求过于频繁可能触发 DeepL 的 IP 限制，不过对于每天几十次的用量完全不是问题。项目地址：[OwO-Network/DeepLX](https://github.com/OwO-Network/DeepLX) [53AI](https://www.53ai.com/news/OpenSourceLLM/2024101753942.html)
