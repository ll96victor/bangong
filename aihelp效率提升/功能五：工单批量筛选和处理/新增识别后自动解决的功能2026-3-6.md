# 需求文档：新增识别后自动解决功能

## 一、需求概述

为脚本 `AIHelp工单批量筛选与处理工具-3.3.0.user.js` 新增"BUG自动解决"功能，一键完成筛选BUG标签工单并自动解决的完整流程。

---

## 二、原脚本信息

- **脚本路径**：同级目录 `AIHelp工单批量筛选与处理工具-3.3.0.user.js`
- **当前版本**：3.3.0
- **UI结构**：悬浮图标（聚合图标）+ 点击展开的功能面板（3x4网格，11个按钮）+ 日志面板
- **已有功能**：筛选BUG标签、筛选MCGG标题、筛选s57描述、清除筛选、批量处理、日志、批量分配（L/N/W/X）、批量解决

## 三、功能需求

### 3.1 新增功能：BUG自动解决

#### 3.1.1 功能描述

用户点击"BJ"按钮后，脚本自动完成以下操作：

**第一阶段：筛选BUG标签工单**
1. 点击"筛选"按钮
2. 点击"重置"按钮
3. 滚动弹窗到底部
4. 点击"包含其中任一标签"输入框
5. 输入"AI识别为BUG bug identified by ai"
6. 点击下拉选项
7. 点击"筛选"按钮

**第二阶段：自动解决流程**
8. 等待1秒（等待筛选结果加载）
9. 点击全选按钮（勾选当前页所有工单）
10. 点击"编辑"按钮
11. 等待批量编辑弹窗加载
12. 点击"工单状态"输入框
13. 在搜索框输入"已解决"
14. 点击下拉选项
15. 点击"内部回复"富文本框，输入配置的回复内容
16. 点击"发送奖励"输入框
17. 点击"15 ProjectCreated.mail"选项
18. 点击"提交"按钮完成操作

#### 3.1.2 操作映射表

| 按钮显示 | 功能名称 | 日志标签 |
|---------|---------|---------|
| BJ | BUG自动解决 | AI识别为BUG自动解决 |

#### 3.1.3 操作前置条件

- 页面已加载完成
- 当前在工单列表页面（URL包含 `tasks?searchType`）
- 不在工单详情页（URL不包含 `ticket`）

## 八、技术实现要点

### 8.1 元素定位参考

| 步骤 | 元素定位方式 |
|-----|------------|
| 筛选按钮 | `button` 包含 `i.el-icon-search` |
| 重置按钮 | `button` 文本为"重置" |
| 标签输入框 | `input[placeholder="请选择标签"]`，通过label包含"包含其中任一标签"定位 |
| 下拉选项 | `li` 元素，匹配标签文本 |
| 筛选确认按钮 | `ToolUtil.clickFilterButton()` |
| 全选复选框 | `span.el-checkbox__inner` |
| 编辑按钮 | `ToolUtil.clickByText(['button', 'span'], '编辑')` |
| 工单状态输入框 | 通过 `.el-form-item__label` 包含"工单状态"定位 |
| 搜索框 | `input[placeholder="搜索"]`，选择距离最近的 |
| 内部回复富文本框 | 通过 `.el-form-item__label` 包含"内部回复"定位，参考富文本输入逻辑 |
| 发送奖励输入框 | 通过 `.el-form-item__label` 包含"发送奖励"定位 |
| 奖励选项 | `span` 元素，匹配"15 ProjectCreated.mail" |
| 提交按钮 | `button` 元素，文本为"提交" |

### 8.2 富文本输入参考

内部回复是富文本框，需要参考脚本 `工单助手与Task客服信息提取合并版 6.4.1 批量筛选模式检测.js` 中"模块 D：提交后自动回复"的逻辑：

```javascript
// 富文本输入示例
const richTextEditor = document.querySelector('.ql-editor');
if (richTextEditor) {
    richTextEditor.focus();
    await ToolUtil.sleep(200);
    // 使用 execCommand 或直接设置 innerHTML
    document.execCommand('insertText', false, config.internalReply);
}
```

### 8.3 等待时间建议

| 操作 | 建议等待时间 |
|-----|------------|
| 点击筛选后等待弹窗 | 800ms |
| 点击重置后 | 500ms |
| 滚动弹窗后 | 500ms |
| 点击输入框后等待搜索框 | 800ms |
| 输入后等待下拉列表 | 800ms |
| 筛选确认后等待结果 | 1000ms |
| 点击编辑后等待弹窗 | 1500ms |
| 点击选项后 | 500ms |

### 8.4 错误处理

- 未找到BUG标签工单时，记录警告日志并提示
- 未找到"已解决"选项时，记录错误日志并提示
- 未找到内部回复输入框时，记录错误日志
- 提交失败时，记录错误日志
