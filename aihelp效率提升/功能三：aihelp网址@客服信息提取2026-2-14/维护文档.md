## 功能列表

### 1. 复制 URL@客服（📋按钮）

**功能描述**：点击按钮复制当前页面链接和客服信息

**实现逻辑**：
1. 提取页面顶部客服名称（格式如 "CN-xxx"）
2. 组合 URL 和客服名称
3. 复制到剪贴板

**关键代码位置**：
- `extractTicketAgentInfo()` - 提取客服信息
- `handleCopyAction()` - 处理复制逻辑

### 2. 更改分组（📁按钮）

**功能描述**：一键将工单分组更改为"CN 二线-BUG"

**实现逻辑**：
1. 检查当前分组是否已是目标分组（跳过已匹配）
2. 点击分组按钮（通过 `icon-ai-group` 图标定位）
3. 等待分配界面弹出
4. 点击客诉队列输入框
5. 输入"CN 二线-BUG"
6. 选择匹配的选项
7. 点击确认按钮

**关键代码位置**：
- `findGroupButton()` - 定位分组按钮
- `waitForQueueInput()` - 等待客诉队列输入框
- `findDropdownOptionFast()` - 快速查找选项
- `handleChangeGroup()` - 主流程

**DOM 结构参考**：
```html

    <svg class="svg-icon icon-ai-group">...</svg>
    CN 二线-BUG
</button>

<!-- 分配界面 -->

        <span>CN 二线-BUG</span>
    </li>

```

## 核心技术实现

### 1. 快速响应 + 重试机制

**设计理念**：快速尝试 → 失败则重试 → 最多3次

```javascript
const CONFIG = {
    dialogWaitTime: 300,      // 弹窗等待（快速响应）
    dropdownWaitTime: 200,    // 下拉框等待（快速响应）
    inputWaitTime: 100,       // 输入等待（快速响应）
    maxRetries: 3,            // 最大重试次数
    retryDelay: 500           // 重试间隔
};
```

**优势**：
- 正常情况快速响应
- 网络波动时自动重试
- 不牺牲用户体验

### 2. 油猴脚本沙箱环境处理

**问题**：油猴脚本中 `view: window` 会报错

**解决**：移除 `view` 属性

```javascript
// 正确写法
element.dispatchEvent(new MouseEvent('click', {
    bubbles: true,
    cancelable: true
    // 不写 view
}));
```

### 3. Vue/React 双向绑定突破

**问题**：直接修改 `input.value` 无法触发框架响应

**解决**：使用原生 setter

```javascript
function simulateInputValue(element, value) {
    const nativeSetter = Object.getOwnPropertyDescriptor(
        window.HTMLInputElement.prototype, 'value'
    ).set;
    nativeSetter.call(element, value);
    element.dispatchEvent(new Event('input', { bubbles: true }));
}
```

### 4. UI 按钮合并拖动

**设计**：所有按钮放入同一容器，拖动逻辑绑定到容器

```javascript
// 主容器
<div class="ai-btn-main-container">
    <div class="ai-action-btn ai-btn-group">📁</div>
    <div class="ai-action-btn ai-btn-tag">🏷️</div>
    <div class="ai-copy-icon-btn">📋</div>
</div>
```

**特性**：
- 三个按钮一起拖动
- 点击按钮不触发拖动
- 位置保存到 localStorage

## 常见问题排查

### 1. 点击按钮无反应

**可能原因**：
- 元素未找到
- 等待时间不够
- 网络波动

**排查步骤**：
1. 打开控制台查看日志
2. 检查是否有 `[AiHelp Ticket Debug]` 开头的日志
3. 查看是否有重试日志

### 2. 下拉框选项未找到

**可能原因**：
- 选项加载延迟
- 选择器不匹配

**排查步骤**：
1. 检查 DOM 结构是否变化
2. 确认 `.el-select-dropdown__item.selected.hover` 是否存在
3. 检查选项文本是否匹配

### 3. 标签添加失败

**可能原因**：
- 标签已存在
- 标签容器未找到
- 选项未匹配

**排查步骤**：
1. 检查 `showtags` 属性
2. 确认 `.elp-cascader__suggestion-item` 是否存在
3. 检查标签文本是否正确
