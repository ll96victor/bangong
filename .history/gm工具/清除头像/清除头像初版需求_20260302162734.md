# 清除头像冷却时间 - 油猴脚本需求文档

## 1. 背景与问题

在工单界面处理工单时，经常遇到需要清除玩家头像冷却时间的情况。当前操作流程繁琐，需要手动在多个页面间切换并输入信息，效率较低。

## 2. 当前操作流程

### 2.1 步骤概览

| 步骤 | 操作位置 | 操作内容 |
|------|----------|----------|
| 1 | 工单列表界面 | 复制玩家账号信息（UID、ServerID） |
| 2 | GM工具页面 | 输入ServerID，选择匹配项 |
| 3 | GM工具页面 | 滚动找到"清除上传头像倒计时"选项 |
| 4 | GM工具页面 | 输入UID，点击执行按钮 |
| 5 | GM工具页面 | 等待结果确认 |

### 2.2 详细操作说明

**第一步：提取账号信息**
- 在工单列表界面复制玩家的账号信息
- 示例：`UID = 1186053970, ServerID = 13814`

**第二步：打开GM工具**
- 访问地址：`https://gm.moba.youngjoygame.com:8090/#/customer/banTool`（可能遇到的情况：首次需要用户登录。如果用户没有登录就进不去，这里需要做一定的保护措施，b）
- 点击左上角的服务器ServerID文本框
- 输入ServerID（如 `13814`）
- 在弹出的匹配项中点击对应选项（有时需手动筛选唯一匹配项）

**第三步：定位功能入口**
- 滚动鼠标滚轮下滑
- 找到"清除上传头像倒计时"选项

**第四步：执行清除操作**
- 点击"清除上传头像倒计时"下方的文本框
- 输入UID对应的值（如 `1186053970`）
- 点击"执行"按钮

**第五步：确认结果**
- 等待网页提示"loading"
- 出现文本 `Result:Ok` 则代表成功

## 3. 自动化需求

### 3.1 功能需求

生成一个油猴脚本，实现用户操作的自动化：

1. **触发方式**：用户点击脚本状态栏时执行
2. **信息提取**：从工单列表界面提取玩家账号信息（UID和ServerID）
3. **自动操作**：在后台/小窗口打开GM工具网页，自动完成所有操作步骤

### 3.2 技术需求

| 需求项 | 要求 |
|--------|------|
| 脚本类型 | 油猴脚本（Tampermonkey） |
| 触发机制 | 点击状态栏触发，非点击时不执行（避免内存占用） |
| 页面适配 | 仅在工单列表界面运行 |
| 信息提取 | 点击后立即提取，无需用户额外输入 |

### 3.3 约束条件

| 约束 | 说明 |
|------|------|
| 运行页面 | 仅工单列表界面 |
| 工单界面地址 | `https://ml-panel.aihelp.net/dashboard/#/manual/tasks?searchType=7&SmartViewID=25285&SmartViewType=4` |
| 触发时机 | 用户点击状态栏时 |
| 执行方式 | 点击后立即提取账号信息 |

### 4.3 关键技术点

| 技术点 | 实现方案 |
|--------|----------|

## 5. 预期输出

### 5.1 成功场景

1. 用户点击状态栏
2. 脚本自动提取UID和ServerID
3. 脚本自动打开GM工具并完成操作
4. 显示成功提示："清除头像冷却时间成功"

### 5.2 异常场景处理

| 异常场景 | 处理方式 |
|----------|----------|
| 无法提取账号信息 | 提示"未找到玩家账号信息" |
| ServerID匹配项有多个 | 提示"存在多个匹配项，请手动确认" |
| 执行失败 | 提示具体错误信息 |
| 网络超时 | 提示"请求超时，请重试" |
